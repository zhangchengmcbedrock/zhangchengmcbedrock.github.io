常见问题
VM 有哪些要求？
VM 至少需要 8GB 的 RAM 和至少 70GB 的磁盘空间。

这些 VM 的用户密码是什么？
没有为用户帐户设置密码。但是，某些软件（尤其是用于远程连接到 VM 的软件）可能需要密码。在这些情况下，您需要先为用户帐户设置密码，然后再使用该软件。

是否可以激活这些映像中的 Windows 许可证以供长期使用？
否，这些 VM 映像使用 Windows 企业评估版，不支持使用产品密钥激活。

是否可以获取 ARM 版本的 VM？
不幸的是，我们目前没有可用的 ARM 版本。我们知道这可能是令人失望的消息，但我们没有任何短期计划来创建这些。但是，我们始终乐于听取用户的反馈和建议，并在规划未来的更新时将其考虑在内。

在Virtualbox上使用这些VM时，我看到了奇怪的渲染怪癖。已知问题？
是的，我们注意到在使用VirtualBox运行这些开发人员映像时存在一些渲染怪癖。“开始”菜单的外观也可能与预期不同。我们目前正在调查此行为。同时，我们感谢您的耐心和理解。

我已在评估窗口中下载并启动了 VM，但 VM 声称评估许可证已过期。帮助？
假设映像在评估窗口中运行，Windows 最终应自行修复。要立即修复它，请运行 Windows 激活疑难解答。
