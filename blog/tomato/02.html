=== 番茄花园 Windows Server 2003 4 in 1 2.1 ===

一、 软件介绍

本安装程序是ISO可启动映像文件，推荐下载以后直接用NERO
刻录映像文件,然后用光盘安装，当然也可以硬盘安装，DOS
下执行I386\winnt.bat即可。

SAT = Windows Server 2003,Standard Edition(番茄花园修改版)

ENTM = Windows Server 2003,Enterprise Edition(MSDN原版未改)

STAM = Windows Server 2003,Standard Edition(MSDN原版未改)


四、安装说明

1.光盘安装(推荐)

请用刻录软件，选择 映像刻录 方式来刻录ISO文件，刻
录之前请先校验一下文件的准确性，刻录速度推荐24X！

a.自动安装
  安装请设置光盘启动，然后选择第1个就可以自动安装！

b. 手动安装
  安装请设置光盘启动，然后选择第2个就可以手动安装！

2. 硬盘安装(支持自动无人职守安装)

a.自动安装
  请提取ISO中I386中的所有内容到硬盘，然后DOS下进入
  I386目录运行winnt.bat就可以安装 。
 
b. 手动安装
  请提取ISO中I386中的所有内容到硬盘，然后DOS下进入
  I386目录运行winnt.exe就可以安装 。安装之前需要先
  加载smartdrv.exe(I386中有)

六、DOS安装指南

推荐解压I386目录到硬盘根目录，然后运行 I386\WINNT.BAT自
动安装。如果I386是在子目录，那么运行WINNT.BAT以后会出现
.\i386而不动了，着个不是问题，只要把.\i386改成I386的实
际路径就可以了！
比如：d:\windows\ent\i386，那么运行WINNT.BAT以后把.\i386
改成d:\windows\ent\i386就可以继续安装了。
